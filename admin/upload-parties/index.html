Firestore Security Rules

Before uploading, set these rules in Firebase Console → Firestore → Rules: 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

After upload, you can restrict writes but keep reads open.
